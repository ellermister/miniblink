前言

miniblink使用了wke的接口。wke的相关介绍可以google一下。 总的来说，miniblink的接口是纯C导出，只要使用wke.h即可加载。无需.lib。

API

unsigned int wkeVersion();

获取目前api版本号
参数：无

CreateFileSystem

CreateFileSystem用于创建新的文件系统。

调试

前往【API Explorer】在线调试，API Explorer 提供在线调用 API、动态生成 SDK Example 代码和快速检索接口等能力，能显著降低使用云 API 的难度，强烈推荐使用。
